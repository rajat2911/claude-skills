# Claude Skills

A collection of useful Claude Code skills.

## Install

```bash
claude /plugin install rajatverma/claude-skills
```

## Skills

### `caveman-mode`

Switches Claude to extreme brevity mode. No filler, no fluff — grunt-level clarity.

**Usage:** Type `/caveman-mode` in Claude Code.

**Example:**
```
/caveman-mode start   # enable
/caveman-mode stop    # disable
```
